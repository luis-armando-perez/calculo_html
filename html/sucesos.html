Coeficiente Binomial

El coeficiente binomial es una forma de elegir k elementos de un conjunto de n elementos, donde el orden no importa y no hay repetición. Se usa para calcular el número de combinaciones posibles. El símbolo se lee como "n sobre k".

$$\binom{n}{k} = \frac{n!}{k!(n-k)!}$$

Experimento Aleatorio

Un experimento aleatorio es un proceso que produce un resultado impredecible. Aunque no se puede saber el resultado exacto de antemano, sí se pueden conocer todos los resultados posibles. El lanzamiento de un dado o una moneda son ejemplos clásicos de experimentos aleatorios.

$$\Omega = \{r_1, r_2, r_3, \dots, r_n\}$$
